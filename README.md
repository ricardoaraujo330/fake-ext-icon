# ChatGPT Atlas fake extension

Extensao local MV3 que so faz uma coisa: ao clicar no icon abre `atlas://extensions`.

O icon foi exportado do bundle nativo do Atlas em:

- `/Applications/ChatGPT Atlas.app/Contents/Resources/Aura_AuraDesignSystem.bundle`
- asset: `Aura/Icons/New/Medium/extensions`

## Carregar no browser

1. Abre `chrome://extensions` ou `brave://extensions`.
2. Ativa `Developer mode`.
3. Clica em `Load unpacked`.
4. Escolhe esta pasta: `/Users/Ricardo/Desktop/Extensions/chatgpt-atlas-fake`
